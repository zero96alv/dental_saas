Información

Los módulos son las secciones principales del menú del sistema. Cada módulo puede contener múltiples elementos de submenú.

  • Orden: Define la posición del módulo en el menú
  • Icono: Clase CSS de Font Awesome (ej: fas fa-users)
  • URL Pattern: Patrón de URL base para el módulo
  • Estado: Activo/Inactivo determina si se muestra en el menú
